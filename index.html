Portfolio

Mini Calculator

The Mini Calculator is a simple yet functional calculator application developed using Python. This project demonstrates the basic principles of arithmetic operations and provides a user-friendly interface to perform calculations.

Features:

  • Basic Arithmetic Operations: The calculator supports addition, subtraction, multiplication, and division.
  • User Input: Users can input two numbers and select the desired operation.
  • Error Handling: The calculator includes error handling for division by zero and invalid inputs.

Technologies Used:

  • Python: The core programming language used to develop the calculator.
  • Tkinter: A standard Python library used to create the graphical user interface (GUI).

Project Objectives:

  • To practice and enhance Python programming skills.
  • To understand and implement a basic GUI using Tkinter.
  • To develop problem-solving skills by handling different arithmetic operations and exceptions.

How It Works:

  1. User Interface: The calculator has a simple GUI with buttons for digits (0-9), operations (+, -, *, /), and a display area to show the input and results.
  2. User Interaction: Users enter numbers and select operations by clicking the corresponding buttons.
  3. Calculation: The program processes the input and performs the selected operation.
  4. Result Display: The result is displayed in the designated area of the GUI.

Hostel Information System

The Hostel Information System is a comprehensive application designed to manage and streamline hostel operations. It provides functionalities to handle student information, room allocation, and other administrative tasks efficiently.

Features:

  • Student Management: Manage student details including personal information, room assignments, and fee status.
  • Room Allocation: Allocate and track room assignments for students, ensuring optimal utilization of hostel space.
  • Reports: Generate detailed reports on student occupancy, and other administrative data.

Technologies Used:

  • PHP: The server-side scripting language used for the backend logic.
  • MySQL: The relational database management system used for storing data.
  • HTML, CSS, JavaScript: Technologies used to build the frontend interface.

Project Objectives:

  • To develop a robust system for managing hostel operations.
  • To enhance PHP and MySQL development skills.
  • To understand and implement database management using MySQL.

How It Works:

  1. Admin Interface: Administrators can log in to the system to manage student records, room allocations, and fees.
  2. Student Registration: Students are registered in the system with their personal details and assigned rooms.
  3. Room Allocation: The system allocates rooms based on availability and maintains a record of room assignments.

BookHaven

BookHaven is an online platform designed for buying used books. This system focuses on the niche market of used books, providing a personalized user experience while adhering to industry standards.

Features:

  • User Registration and Authentication: Secure user accounts for buying books.
  • Book Browsing and Searching: Browse and search books by genre, author, or category
  • Shopping Cart Functionality: Easily add books to the cart and make purchases.
  • Order Management: Track orders and view order history.
  • Administrative Tools: Admins can manage books, users, and orders to ensure smooth operation of the platform.

Technologies Used:

  • PHP: Server-side scripting language used for backend development.
  • MySQL: Relational database management system for storing user and book data.
  • HTML, CSS, JavaScript: Technologies used to build the frontend interface.

Project Objectives:

  • To create an efficient system for buying and selling used books online.
  • To enhance PHP and MySQL development skills.
  • To develop an admin panel for managing platform content,user account and order tracking functionalities.

How It Works:

  1. User Interface: Users can browse and search for books through an intuitive interface.
  2. User Registration: Users sign up and log in to access additional features like searching books and making purchases.
  3. Shopping Cart: Users can add books to their cart and proceed to checkout for purchase.
  4. Order Management: The system manages orders, updates stock status, and provides order tracking and history.
  5. Admin Panel: Admins can add, edit, and delete book listings, manage user accounts, and oversee order processing.
